# Global_warming
1. Программа парсирующая информацию.
2. Советы для предотвращения глобального потепления.
3. Как началось потепление и его последствия.

Тип проекта: сайт на базе sait_portfolio.

Чек-лист:
1. Проверить работоспособность кнопок в вверху сайта. Проверить внешний вид сайта.
2. Функциональность: работа кнопок. Интерфейс: правильное рассположение текста и изображений.
3. Результаты.Фунциональность: кнопки переносят на нужную часть сайта. Интерфейс: текст или изображения не налазят друг на друга, не слишком большое расстояние между текстами и изображениями.

Что пока что не готово: в некоторых местах слишком большое расстояние между текстами, пока что нет раздела с источниками ифнормации.
